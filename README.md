## Installation

### Using Git
```
git clone https://github.com/fr0st-iwnl/XPicker.git
```

## Usage

`XPicker` lets you capture colors from your screen effortlessly. Right-click to copy HTML color codes, hold Ctrl for RGB colors, and press `ESC` to exit.
